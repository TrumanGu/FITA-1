活动和荣誉

    概况：

  • 南京工业大学FITA社是在校团委的领导下，由学校计算机科学与技术学院主管的，由一群对新鲜技能、外部企业有着好奇感的爱好者组成。

    • 目的：

  • (一）丰富学生的课余生活，为对IT技术感兴趣的学生们提供学习机会。架起一座走向前沿技术、接触外部新鲜资讯的桥梁，让学生真实地去接触尖端企业与技术，开阔眼界，学习知识。
  • (二）培养出一些具有较高技术水平的同学，让他们能通过这个平台接受一些软件开发的项目，也为相关同学日后的就业打下坚实的基础。
  • (三）通过部分同学在ＩＴ方向的成长，带动全校的信息化建设氛围，迎接互联网大潮。

    • 宗旨：

  • 技术、创新、交流、服务。
  • 旨在促进大学生对IT技术的了解，并鼓励创新。

  • Stack Overflow是一个私营网站，是Stack交换网络的旗舰网站,由Jeff Atwood和Joel Spolsky于2008年创建。它被创建为一个更加开放的替代早期问答网站，如Experts-Exchange。该网站的名称是在2008年4月由Atwood的热门节目博客Coding Horror的读者投票选出的。它包含有关计算机编程中广泛主题的问题和答案。

    该网站作为用户提问和回答问题的平台，并通过会员和积极参与，向上或向下投票问题和答案，并以类似于维基或Digg的方式编辑问题和答案。Stack Overflow的用户可以获得声望点和“徽章”;例如,一个人获得10点声望，用于接收“向上”上给出的问题的答案和“向上”问题的表决 票5分，也可以接收徽章的价值贡献，其中代表了传统问答网站的一种游戏化。用户可以通过增加声誉来解锁新权限，例如投票，评论甚至编辑其他人帖子的能力。所有用户生成的内容均根据知识共享属性 - ShareAlike许可进行许可。

    结束问题是Yahoo!的主要区别。答案和一种防止低质量问题的方法。该机制于2013年进行了彻底改革; “暂停”后编辑的问题现在出现在审核队列中。杰夫阿特伍德在2010年表示，重复的问题并不是一个问题，而是如果这些额外的问题通过在搜索引擎中增加相关的关键字匹配来增加网站的额外流量，它们就构成了一个优势。

    截至2018年8月，Stack Overflow拥有超过9,000,000注册用户，并且在2018年中期超过了16,000,000个问题。根据分配给问题的标签类 型，网站上讨论最多的八个主题是：Java，JavaScript，C＃，PHP，Android，jQuery，Python和HTML。

    Stack Overflow还有一个Jobs部分，可以帮助开发人员找到下一个机会。对于雇主而言，Stack Overflow提供工具来标记他们的业务，在网站上公布他们的空缺，并从Stack Overflow的开发人员数据库中找到可供联系的候选人。

  • GitHub Inc.是一个基于Web的托管服务，用于使用Git进行版本控制。它主要用于计算机代码。它提供了Git的所有分布式版本控制和源代码管理（SCM）功能，以及添加自己的功能。它为每个项目提供访问控制和一些协作功能，如错误跟踪，功能请求，任务管理和维基。

    GitHub提供私有存储库和免费帐户的计划，这些计划通常用于托管开源软件项目。截至2018年6月，GitHub报告拥有超过2800万用户和5700万个存储库（包括2800万个公共存储库。），使其成为世界上最大的源代码主机。

  • CSDN即“中国的软件开发者网络”或“中国软件开发者网络”，属于白莲Midami数码科技有限公司，是国内最大的网络之一，软件开发者在中国。CSDN提供Web论坛，博客托管，IT新闻和其他服务。CSDN拥有约1000万注册用户，是中国最大的开发者社区。

  • freeCodeCamp（也称为“ Free Code Camp ”）是一个非盈利组织，由一个互动学习网络平台，一个在线社区论坛，聊天室，媒体出版物和打算学习网络开发的本地组织组成。任何人都可以访问 从向学生介绍HTML，CSS和JavaScript的教程开始，学生可以进入项目作业，他们必须单独或成对完成。完成所有项目任务后，学生将与其他非营利组织合作构建Web应用程序，为学生提供实际的开发经验。

  • HackerRank是一家技术公司，专注于消费者和企业的竞争性编程挑战，开发商通过尝试根据提供的规范进行竞争来进行竞争。 HackerRank的编程挑战可以用各种编程语言（包括Java，C++，PHP，Python，SQL，JavaScript）解决，并跨越多个计算机科学领域。

    在消费者方面，当程序员向编程挑战提交解决方案时，他们的提交将根据其输出的准确性进行评分。然后，程序员在HackerRank排行榜上排名全球，并根据他们的成就获得徽章，以推动用户之间的竞争。除了个别编程挑战之外，HackerRank还举办竞赛（通常被HackerRank称为“CodeSprints”），其中用户在一段时间内竞争相同的编程挑战，然后在事件结束时进行排名。HackerRank是竞争性计算机编程中不断增长的游戏化趋势的一部分，其网站的消费者端可供编码人员免费使用。
